
# SparkAI - Find Your Frequency

SparkAI is a next-generation dating platform powered by Google Gemini and secured with hardware-level encryption.

## Features
- **AI Matchmaking**: Real-time compatibility analysis using Gemini 3.
- **Grounded Planning**: Date suggestions powered by Gemini 2.5 + Google Maps.
- **E2EE Messaging**: Message privacy via RSA-2048 encryption.
- **Voice Coach**: Real-time "Wingman" AI for conversation practice.

## Setup
1. **Gemini API**: Add `API_KEY` to your environment.
2. **Database**: Run the SQL script in `components/ProductionChecklist.tsx` within your Supabase SQL editor.
3. **Deploy**: Optimized for Vercel. Use framework "Other".
